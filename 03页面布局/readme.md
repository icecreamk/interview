## 页面布局
- 三栏左右宽度固定，中间自适应
- 五种实现方式：./index.html
    * float
        * 优：兼容性好
        * 缺：需清楚浮动
        * 不适应高度不固定
    * position
        * 优：快捷
        * 缺：脱离文档流，子元素也会脱离文档流
        * 不适应高度不固定
    * flexbox
        * 优：解决以上方案缺点
        * 缺：兼容性不好
        * 适应高度不固定
    * table
        * 优：兼容性好
        * 缺：三栏高度会同步变化
        * 适应高度不固定
    * grid
        * 优：新技术，实现复杂布局且代码量小
        * 缺：兼容性不好
        * 不适应高度不固定

## 常见题目
#### 三栏布局
- 左右宽度固定，中间自适应
- 上下高度固定，中间自适应

#### 两栏布局
- 左宽度固定，右自适应
- 右宽度固定，左自适应
- 上高度固定，下自适应
- 下高度固定，上自适应
